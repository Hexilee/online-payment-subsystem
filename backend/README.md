## 在线支付系统-订单管理子系统



#### 前端 frontend

先安装 yarn 和 parcel

```bash
# 安装依赖
> yarn install
```

```bash
# 直接运行
> yarn dev

# 或打包
> yarn build
```



#### 后端 backend

安装 python3

```bash
# 创建虚拟环境
> python3 -m venv dev
```

```bash
# 进入虚拟环境
> source dev/bin/activate
```

```bash
# 安装依赖
(dev)> pip3 install -r requirements.txt
```

```bash
# 写配置
(dev)> mv config_sample.py config.py
(dev)> vim config.py
```

```bash
# 运行
(dev)> python3 main.py
```
